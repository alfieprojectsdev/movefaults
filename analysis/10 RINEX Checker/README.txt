RINEX2 Data Checker v. 1.1

Created by kurt, May 2024
Last updated: November 2024


Description:
This family of Python scripts checks for RINEX2 data completeness by listing the days with and without a RINEX2 file.
Input: RINEX2 O files
Output: .TXT file

Scripts:
RINEX2data_checker.py = to check for a particular station in a given year
RINEX2data_stations_checker.py = to check for all the stations in a given year

How to use:
1. Download and copy the scripts to any folder on your device.
2. Copy the path of your RINEX folder. Its subfolders will also be checked.
3. Run your needed script by double clicking the .py file. Input RINEX folder path.
4. The output .txt file will be in the same folder as the script.
5. Repeat steps 2-3 as needed. 
 
Note: You may replace 'rinex2_folderPath' with the folder path of your RINEX2 data.
